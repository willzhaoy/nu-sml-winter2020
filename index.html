Content
What is this course about? This course introduces statical machine learning methods from a theoretical perspective. Topics include the maximum likelihood inference, regularization principle, risk minimization framework, cross-validation, high dimensional inference and nonparametric methods.
Prerequisites: COMP_SCI 214: Data Structures and Data Management
Reading: There is no required textbook for this class, and you should be able to learn everything from the lecture notes and homeworks.
